# IntoTheNight

Projekt Roblox zarządzany przez Rojo.

## Jak uruchomić:

1. Otwórz terminal w tym folderze
2. Uruchom: `rojo serve`
3. Otwórz Roblox Studio
4. Kliknij plugin Rojo i "Connect"

## Struktura:

- `src/ServerScriptService/` - Skrypty serwerowe
- `src/ReplicatedStorage/` - Moduły współdzielone
- `src/StarterPlayer/` - Skrypty klienta
- `src/Workspace/` - Obiekty w świecie gry
